Node Version Manager
====================

## install
> wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.3/install.sh | bash

## usage

nvm install stable
nvm uninstall 0.12

nvm ls
nvm ls-remote

nvm current
nvm use 0.12
nvm use system
nvm cache clear

nvm alias default node
nvm alias default 4.2.1

nvm install iojs stable
nvm install node
