Yarn-javascript package manager
===============================

## 安装
npm -g i yarn

## 使用
yarn init -y "开始新项目"
yarn add package|package@version|package@tag --dev "添加依赖包"
yarn upgrade package|package@version|package@tag "升级依赖包"
yarn remove package "移除依赖包"
yarn install "执行安装"
yarn list "列出安装的包"
yarn outdated "过期的包"
yarn info package "查看包信息"
